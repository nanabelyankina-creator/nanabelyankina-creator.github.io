# Изображения для сайта клиники

Добавьте следующие файлы для оптимальной загрузки:

## Hero (главная страница)
- `hero.avif` — баннер (формат AVIF)
- `hero.webp` — баннер (формат WebP)
- `hero.jpeg` или `hero.jpg` — баннер (fallback JPEG)

Если файлов нет, используется внешнее изображение с Unsplash.

## Аватар врача по умолчанию
- `doctor-placeholder.avif`
- `doctor-placeholder.webp`
- `doctor-placeholder.jpeg`

Плейсхолдер для врачей без загруженного аватара.
