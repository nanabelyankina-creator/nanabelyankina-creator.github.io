# IT-TENDER diploma site

## Stack
- PHP (native)
- MySQL
- HTML/CSS/JS

## Setup
1. Create virtual host, project path: `c:/OSPanel/home/diplom-app/ittender`.
2. Import SQL file `database.sql` into MySQL.
3. Check DB credentials in `config.php`.
4. Open in browser:
   - Public: `http://ittender/` (or your host)
   - Admin: `http://ittender/admin/login.php`
